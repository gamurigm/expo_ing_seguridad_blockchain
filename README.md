# Blockchain Empresarial: Tecnología, Cadenas y Regulación

Material académico de investigación, soporte y defensa de exposición para la asignatura de **Ingeniería en Seguridad** (Unidad 2).

Este repositorio está estructurado y organizado metódicamente para brindar un respaldo científico inquebrantable a cada sección temática de la exposición, facilitando una defensa rigurosa e incontestable ante el tribunal académico.

---

## 📂 Estructura del Repositorio

El proyecto ha sido organizado meticulosamente en subdirectorios temáticos para facilitar la navegación y garantizar un orden profesional:

```
/ (raíz)
├── README.md                          # Guía de inicio y mapa de navegación rápido (este archivo)
├── toc_expo_blockchain.md             # Índice interactivo que mapea cada subtema a sus PDFs de respaldo
├── documentacion/                     # Guías de estudio, guiones técnicos y glosarios
│   ├── estructura_expo_blockchain.md  # Guion y desglose técnico detallado de la exposición
│   ├── glosario_tecnico_blockchain.md # Glosario de acrónimos y conceptos avanzados de seguridad
│   └── analisis_documentos_blockchain.md # Análisis en profundidad y síntesis cruzada de los papers clave
└── bibliografia/                      # Biblioteca digital con 20 papers académicos y whitepapers originales
    ├── 1_fundamentos/                 # PDFs de respaldo sobre DLT, Bitcoin y seguridad base
    ├── 2_cadenas/                     # PDFs de Ethereum, Solana, Avalanche, Polygon, Fabric y consensos
    ├── 3_aplicaciones/                # PDFs sobre Ripple, Smart Contracts, Trade Finance y ZKPs
    ├── 4_tokenizacion_defi/           # PDFs de tokenización de activos RWA, AMMs (Uniswap) y RTGS/LSM
    └── 5_regulacion_casos/            # PDFs sobre regulación MiCA, stablecoins 2.0 y oráculos de reserva
```
